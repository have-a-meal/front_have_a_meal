개발자 : 이재현


프로젝트 한줄소개 : 학교에서 키오스크에서 구매하는 학식권을 모바일에서 구매하고 이를 인증할 수 있는 프로그램


프로젝트 화면 구조
- 로그인 / 회원가입
- 날짜별 식단표 출력
- 식권 구매
- 구매한 식권을 QR로 표시
- 결제 내역 표시
